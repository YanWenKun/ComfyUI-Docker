# 用于 ComfyUI 的 Docker 镜像

中文 | ℹ️ *link:README.adoc[English]*

https://github.com/comfyanonymous/ComfyUI[ComfyUI]
是一套使用节点工作流的 AIGC 图形界面。 +
本仓库所构建的
https://hub.docker.com/r/yanwk/comfyui-boot[Docker 镜像]
包含其依赖项，及下载／启动脚本。

## 快速开始 - NVIDIA GPU

```sh
mkdir -p \
  storage-models/models \
  storage-models/hf-hub \
  storage-models/torch-hub \
  storage-user/input \
  storage-user/output \
  storage-user/user-profile \
  storage-user/user-scripts

# Add sudo if needed
docker run -it --rm \
  --name comfyui-cu130 \
  --pull=always \
  --runtime=nvidia \
  --gpus all \
  -p 8188:8188 \
  -v comfy-cache/.cache:/root/.cache \
  -v comfy-cache/.config:/root/.config \
  -v comfy-nodes/.local:/root/.local \
  -v comfy-nodes/custom_nodes:/root/ComfyUI/custom_nodes \
  -v "$(pwd)"/storage-models/models:/root/ComfyUI/models \
  -v "$(pwd)"/storage-models/hf-hub:/root/.cache/huggingface/hub \
  -v "$(pwd)"/storage-models/torch-hub:/root/.cache/torch/hub \
  -v "$(pwd)"/storage-user/input:/root/ComfyUI/input \
  -v "$(pwd)"/storage-user/output:/root/ComfyUI/output \
  -v "$(pwd)"/storage-user/user-profile:/root/ComfyUI/user \
  -v "$(pwd)"/storage-user/user-scripts:/root/user-scripts \
  -e CLI_ARGS="--fast" \
  yanwk/comfyui-boot:cu130-slim-v2
```

## CUDA 镜像标签

### Slim

`slim` 镜像仅预装基本的 ComfyUI 与 Manager，同时预装大量依赖项，方便后续无痛安装热门自定义节点（扩展插件）。推荐初学者使用。

* link:cu130-slim-v2/README.adoc[`cu130-slim-v2`] ⭐
** CUDA 13.0, Python 3.13 (GIL 版本), 无 xFormers

* link:cu128-slim/README.adoc[`cu128-slim`]
** CUDA 12.8, Python 3.12

* link:cu126-slim/README.adoc[`cu126-slim`]
** CUDA 12.6, Python 3.12

### MEGAPAK

`megapak` 镜像为整合包，包含开发套件与常用自定义节点（扩展插件）。

* link:cu130-megapak-pt211/README.adoc[`cu130-megapak-pt211`] ⭐
** CUDA 13.0, Python 3.13 (GIL 版本), GCC 14, PyTorch 2.11.0

* link:cu128-megapak/README.adoc[`cu128-megapak`]
** CUDA 12.8, Python 3.12, GCC 14, PyTorch 2.11.0

* link:cu128-megapak-pt29/README.adoc[`cu128-megapak-pt29`]
** CUDA 12.8, Python 3.12, GCC 11, PyTorch 2.9.1

* link:cu128-megapak-pt28/README.adoc[`cu128-megapak-pt28`]
** CUDA 12.8, Python 3.12, GCC 11, PyTorch 2.8.0

* link:cu126-megapak/README.adoc[`cu126-megapak`]
** CUDA 12.6, Python 3.12, GCC 13, PyTorch 2.9.1
